DIO Orchestrator

Distributed Inference Orchestration System

v0.1.0-beta • Work in Progress

Test A: Latency Check

Validates gRPC routing efficiency. Sends a single packet through the Go Manager to a Python Worker and back.

Test B: Load Simulation

Triggers the autoscaler. Simulates a sudden traffic spike to force the Docker Manager to provision new workers.

Test C: Cost & Budget

Audits token usage. Sends a payload to verify token counting logic and budget enforcement policies.
